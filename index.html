v2 Banana
Current · v2.0.0

A modern UI & full PDU support

Stable release · v2.0.0

  • Modern UI in Flet. Material 3 surfaces, responsive layout 880×620 → 4K, sticky bottom bar that respects the Windows taskbar.
  • Polished launcher. Clearer device cards and a Switch Device button always one click away.
  • Full UPS & PDU support. APC Rack PDU command set with phase readings, bank/outlet status, environmental sensors.
  • Smarter recovery. Extended alternate-command fallback table; faster BM> bootloader handling on cold boots.
  • Bug-report shortcut. Copy Diagnostics produces a self-contained blob safe to paste into a ticket.
  • Codename system. Each major release ships under a color name (Aqua, Banana, Cobalt, …).

v1 Aqua
Shipped · v1.x

The first stable foundation

v1.0 → v1.7-beta · WBLS rebrand

  • Modern Windows GUI for APC NMC serial configuration. No terminal required.
  • Side-by-side CLI for headless / automation use.
  • Generated Word reports with branded cover, profile-aware title, optional credentials block.
  • Configuration helpers: TCP/IP, DNS, NTP, system identity, time sync, email recipients.
  • Smart fallback: detbat -all → battstat when the firmware doesn't recognize the primary command.
  • Bootloader recovery: drops into BM> after reboot? Sends boot and waits patiently for the firmware.
  • Manual override dialog when a command times out — send raw commands, copy diagnostics, then resume or abort.
  • Persistent technician name & Advanced Report selection (Email / DNS / NTP optional sections).
  • Verbose logging with rotating file at %APPDATA%/WBLS/nmc_tools/logs/, password redaction, in-app Info / Debug toggle.
  • Launcher screen picks UPS or PDU NMC up front; right command set follows automatically.
  • Quality-of-life polish: COM port placeholder so Bluetooth virtual ports don't hang the connection, validated IP/subnet/gateway fields, native maximize/restore.
