# What's New in v1 — Aqua

The first stable release of the rebranded **WBLS Engineering — NMC Configuration Tools**. Highlights of the journey from v1.0 to v1.x:

- **Modern Windows GUI** for APC NMC serial configuration. No terminal required for field technicians.
- **Side-by-side CLI** (`python -m cli.nmc_cli`) for headless / automation use.
- **Generated Word reports** with branded cover, profile-aware title (UPS or PDU), and optional credentials block.
- **Configuration helpers**: TCP/IP, DNS, NTP, system identity, time sync (UTC+08:00), email recipients.
- **Smart command fallback**: `detbat -all` → `battstat` automatically when the firmware doesn't recognize the primary command.
- **Bootloader recovery**: when the NMC drops into `BM>` after a reboot, the tool sends `boot` and waits patiently for the firmware to come up.
- **Manual override dialog** when a command times out — send raw commands, copy diagnostics, then resume or abort.
- **Persistent technician name** and Advanced Report selection (Email / DNS / NTP optional sections).
- **Verbose logging** with a rotating file at `%APPDATA%/WBLS/nmc_tools/logs/`, password redaction, and an in-app Info / Debug toggle.
- **Launcher screen** lets you pick UPS or PDU NMC up front; the right command set and report layout follow automatically.
- **Quality-of-life polish**: COM port placeholder so Bluetooth virtual ports don't hang the connection, validated IP/subnet/gateway fields, and a clean Windows-native maximize / restore.

Thank you for using the tool — feedback drives the v2 roadmap.
