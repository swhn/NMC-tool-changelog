# What's New in v2 — Banana

A major rewrite focused on professionalism and broader device support.

- **Modern UI in Flet** — Material 3 surfaces, responsive layout that scales from 880×620 to 4K, and a sticky bottom bar that respects the Windows taskbar.
- **Polished launcher** — clearer device choice cards and a Switch Device button that's always one click away.
- **Full UPS and PDU support** — APC Rack PDU command set and a PDU-shaped report (Phase Readings, Bank Status, Outlet Status).
- **Smarter recovery** — alternate-command fallback table is extended and BM> handling is faster on cold boots.
- **Bug-report shortcut** — Copy Diagnostics on the override dialog produces a self-contained blob (version, OS, log tail, last serial buffer) safe to paste into a ticket.
- **Codename system** — each major release ships under a color name (v1 = Aqua, v2 = Banana, v3 = Cobalt, ...).

Full details on the [WBLS NMC Tools changelog](https://nmc-tool.madebysai.com/).
